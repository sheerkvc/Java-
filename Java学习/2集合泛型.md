#### 继承extends：

```
构造方法：public 不能
				private 不能
成员变量：public 能
				private 能（不可直接用
成员方法：添加到虚方法表中(没有被static、final等修饰) 能
				private 不能
```

1. ﻿﻿﻿Java只能单继承：一个类只能继承一个直接父类。

2. ﻿﻿﻿Java不支持多继承、但是支持多层继承。

3. ﻿﻿﻿Java中所有的类都直接或者间接继承于Object类。

> - ﻿﻿有方法重写

> 运行也看左边：java运行代码的时候，实际获取的就是左边父类中成员变量的值
